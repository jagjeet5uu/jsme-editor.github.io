Basic editing

Rings are added connected by a single bond (when not added to a terminal atom). To add a spiro ring, press Shift when adding it. Alternatively — and on touch devices — press the icon; the next ring will then be added as spiro.

With single bond selected, clicking an existing single bond changes it to double.

The chain tool draws chains, saturated rings, and converts single bonds to double bonds on click.

A molecule may be moved by dragging free space, and rotated by holding Shift while dragging.

Stereochemistry

Stereochemistry at C4 centers, double bonds, and allenes is supported. Use the up / down wedge bonds to mark stereochemistry at C4 centers — only bonds with a sharp point towards the atom are considered. When generating SMILES, the editor tries to infer missing stereo features; in unresolvable cases it issues an error and emits SMILES without stereo.

When the autoez option is set, SMILES with E,Z stereochemistry on all non-ring double bonds is generated automatically. Without it (or for ring double bonds), mark a double bond as stereo by clicking it with the stereo-bond button selected — the bond turns magenta.

Stereochemistry may be completely disabled with the nostereo option.

Atom numbering

Atom numbering (marking) is enabled by the number option — default for reaction input. Button appears in the menu. Click it to enter numbering mode, type a number on your keyboard (e.g. 2 or 1 2), then click the atom. To remove a number, press 0 and click the atom.

Input of multipart structures

By default only non-disconnected structures are allowed. Enable multipart input with the multipart option — a button appears in the menu. To add another component: click , select a template (atom, ring, or bond), and click free space in the drawing area. Without clicking New first, clicking free space simply moves or rotates the last touched molecule. In multipart mode, deletes the last touched molecule.

Reaction input

Enable reaction input with the reaction option. Buttons , and appear, plus an arrow in the drawing area. Draw reactants, products, and modulators (modulators above the arrow) using the multipart workflow above. Clicking the arrow button copies the reactant to the product, preserving atom numbering.

Query features

The query button (when enabled) opens a window for building SMARTS atom/bond queries by combining atom attributes. Pick the SMARTS options, then click the target atom or bond.

Copy & paste

Right-click opens the Copy / Paste pop-up — copy the molecule as SMILES or MOL, or paste MOL, SDF, or RXN into the editor.

On supported touch devices, the menu opens by touch-and-hold on the editor surface. Ctrl+C and Ctrl+V work in most browsers.

Drag & drop of files

HTML5-compatible browsers support dropping files into the editor. Since version 2013-03-24, JSME accepts MOL, SDF, and RXN text files.

To drag a molecule out of the editor into another application (Word, text document, desktop), grab the icon in the lower-right corner of the editor.
