The molecule editor built for the web.

JSME is a free, fast molecule and reaction editor that runs in any modern browser — desktop, tablet, or phone. Direct successor to the JME applet, with no plugins required.

Everything you need to sketch chemistry

A focused toolset for drawing molecules, building reactions, and exporting to the formats cheminformatics pipelines actually use.

⌨

Keyboard shortcuts

Built-in substituent menu and shortcuts for fast assembly of large, complex structures.

↔

Reactions & SMIRKS

Draw reactions natively. Export as reaction SMILES or SMIRKS for downstream processing.

⤓

SMILES & Molfile

Canonical SMILES output independent of drawing order. Read and write MDL/Symyx Molfiles.

⌕

Substructure & SMARTS

Build complex substructure queries that translate automatically into SMARTS for database search.

📱

Works on touch

Designed for iPhone, iPad, and Android alongside desktop — no plugin, no install.

📋

Clipboard support

Copy SMILES or Molfile to the system clipboard, paste Molfiles directly into the editor.

Embed it, or grab a release

JSME ships as a self-contained JavaScript library. Drop in two lines, instantiate the applet, and you're done.

Download zip npm package
// 1. Load the JSME loader <script src="jsme/jsme.nocache.js"></script> // 2. Mount when ready <script> function jsmeOnLoad() { new JSApplet.JSME("editor", "380px", "340px"); } </script> <div id="editor"></div>
